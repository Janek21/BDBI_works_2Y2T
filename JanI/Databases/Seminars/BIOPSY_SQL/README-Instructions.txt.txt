1) Open the file BIOPSYDB.sql in SQL developer.
2) Select all lines which say DROP TABLE and press the green arrow in your left corner on the top.
3) Check the output log doesn't give any errors.
4) Select all lines which say CREATE TABLE and press the green arrow.
5) Check the output log doesn't give any errors.
6) Select all lines which said Insert into and press the green arrow.
      If you get an error related with the use of "," in the decimal separator you have to use the script BIOPSYDB_english.sql
      and repeat the previous steps.
7) Select all lines which said ALTER TABLE and press the green arrow.
8) Check the output log doesn't give any errors.
9) Go to Table section, click Refresh button and you have to see all the tables created with data.
